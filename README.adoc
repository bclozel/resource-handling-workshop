in https://spring.io/blog/2014/07/24/spring-framework-4-1-handling-static-web-resources[this blog post].

## Requirements

* Maven 3.2+
* JDK6+
* git

Optional:

* Node.js (see https://github.com/creationix/nvm[nvm])
* gradle (see http://gvmtool.net/[gvm])
